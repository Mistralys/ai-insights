# Persona Quickstart Guide

> How to create your own AI agent persona — from zero to a working prompt file.

Where do you want to use your persona?

| | Where | Best For |
|---|-------|----------|
| 🌐 | [**Web — Gemini Gems, Claude Projects, ChatGPT, Grok…**](./persona-quickstart-web) | Using personas in AI chat interfaces and web apps. No installation required. |
| 💻 | [**IDE — VS Code, Claude Code, Cursor…**](./persona-quickstart-ide) | Using personas directly in your coding environment. |

Both paths produce a well-structured persona file. The web path requires nothing beyond a browser; the IDE path offers deeper integration and the best results through the AI Insights project.
