Catering to multiple platforms

The Scaling Problem


One persona. Many targets.

Each with its own metadata format.


VS Code needs YAML frontmatter.
Claude Code needs a different header.
Deep Agents need plain Markdown.


The persona builder: a templating system
to write once, deploy everywhere.

Maintaining copies by hand doesn't scale. One change to the persona means updating 3 files in 3 different formats. The persona builder solves this with a template engine.
