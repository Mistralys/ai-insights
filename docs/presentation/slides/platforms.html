● Gemini (Web)
With Gems
● Claude (Web & Desktop)
With Project instructions
● VS Code, Cursor, Claude Code
Fully supported (global + project)
● Copilot (Web)
Not yet — manual copy & paste
● ChatGPT (Web)
Project instructions (8k limit) or "GPTs" (paywall)
● Grok (Web)
With Custom Agents (4 max, paywall)
The key point: personas aren't tied to one tool. You can use them almost anywhere. But each platform has its own format for how the instructions are loaded — which is exactly the problem the build system solves.
