Discovering personas

What is an Agent Persona?

A structured instruction document that tells an AI
who it is, what it does, and how it works.


More than a prompt — an operating manual.

A persona is not a one-line system prompt. It's a complete operating manual that defines identity, knowledge, constraints, and workflow. Think of it as the difference between telling someone "be a chef" and handing them a detailed job description with equipment lists, dietary rules, and step-by-step procedures.
