Both paths use identical persona prompts and ledger state — the orchestrator is a different driver, not a different system.

The IDE path is already deterministic — agents hand off automatically through the ledger. The human is involved at Planner, PM, and Synthesis (as shown in the previous slide). The orchestrator removes even that — a LangGraph supervisor drives the entire chain from a single CLI command. Same MCP server, same personas, same ledger. Useful for CI, batch processing, or fully hands-off runs.
