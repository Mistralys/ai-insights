Catering to multiple platforms

One Source, Multiple Outputs

Target conditionals adapt to each platform automatically.

{{#if target_vscode}}
  Invoke `runSubagent` with agentName: "WP Decomposer"
{{else if target_claude_code}}
  Use the `Task` tool with agent: "WP Decomposer"
{{else if target_deep_agents}}
  Use the `task` tool with subagent_type: "wp-decomposer"
{{/if}}

From the Project Manager persona — each platform has a different tool API
for dispatching sub-agents. The template handles it.

Variables {{agent_name}} | Partials {{> shared-partial}} | Conditionals {{#if}}

This is real syntax from the Project Manager persona. VS Code uses runSubagent, Claude Code uses Task, Deep Agents uses task with a slug. One source file, three correct outputs. The template language has just three constructs: variables for values, partials for shared content blocks, and conditionals for platform differences. No programming required.
