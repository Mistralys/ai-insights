Beyond a single persona

Build Your Own Personas

You don't need to start from scratch — let the model help you build it.

1. Pick a task you repeat often
Meal planning, travel research, email drafting, meeting summaries…
2. Give the model the AI Insights' persona creation guide
This guide teaches the structure, principles, and patterns.
3. Describe what you need in your own words
The model generates the persona for you — following the guide.
4. Use it, observe, refine
Personas are living documents. Each use shows you what to tighten.

A Quickstart Guide with the creation guide and step-by-step
instructions are linked at the end.

The key takeaway: you don't need to be a prompt engineer. The AI Insights project includes a Persona Design Guide — a detailed reference document that teaches the model itself how to create well-structured personas. You feed the guide to the model, describe what you want, and it generates the persona for you. Then you use it, see where it falls short, and tighten the instructions. A quickstart guide with everything you need is linked at the end.
